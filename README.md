# SiliconCloud Stats

SiliconCloud Stats 是一个用于分析 SiliconCloud 平台使用情况的工具。通过输入 Cookie，可以拉取 SiliconCloud 控制台 API 来实现各种分析功能，如 token 用量分析等。

## 功能特点

- 通过 Cookie 认证访问 SiliconCloud API
- Token 用量分析
- 明暗主题切换
- 响应式设计，支持移动端
